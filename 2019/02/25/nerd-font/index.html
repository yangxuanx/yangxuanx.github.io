nerd font

2019-02-25

​ 最近在浏览论坛时，发现一个vim启动页插件，配套的还有一个vim中显示文件类型图标插件。但是在普通的字体中是不显示文件图标的，为了显示文件图标，必须配套使用nerd font字体。

​ 实际操作中只需要下载当前使用等宽字体的的nerd版本（例如hack，下载安装hack nerd font后安装即可），安装后即可再vim中显示正常的图标。

​ 值得一提的时，安装nerd字体后，不需要修改系统字体或者终端字体为nerd字体，nerd字体更像是一种补丁的方式。
